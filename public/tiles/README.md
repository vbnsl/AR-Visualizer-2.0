# Tile images

Drop tile images into one of these folders; the app discovers them by folder and uses that as the surface type.

- **`wall/`** — tiles shown only in the Wall viewer
- **`floor/`** — tiles shown only in the Floor viewer  
- **`both/`** — tiles shown in both Wall and Floor pickers

Supported formats: PNG, JPG, JPEG, WebP, SVG, GIF, AVIF.

Refresh the page after adding or removing files to update the catalog.
